## Code example

Here's a simple example of a Lambda function in Python:

```python
import json

def handler(event, context):
    """Main Lambda handler."""
    body = json.loads(event.get("body", "{}"))

    return {
        "statusCode": 200,
        "headers": {"Content-Type": "application/json"},
        "body": json.dumps({
            "message": "Hello from Lambda!",
            "input": body
        })
    }
```
